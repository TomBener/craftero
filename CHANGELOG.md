# Craftero Changelog

## [0.1.0] - Initial Version

### Added

- Local-first Zotero database access with fuzzy search
- Search across titles, authors, tags, DOI, abstracts, and citation keys
- Collection dropdown to filter search results
- Sync Zotero items to Craft collections
- Batch sync all visible search results
- Multiple sync workflows: Sync, Sync & Open in Craft, Sync & AI Summarize
- Sync Zotero notes and PDF annotations
- Daily note linking in Craft for reading dates
- Item management actions: copy keys, delete from Craft, open URL/DOI, open in Zotero
- AI integration via Raycast AI Commands with MCP server support
- Automatic deduplication by Zotero URI
